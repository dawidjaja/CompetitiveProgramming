Faktor Bilangan

Nama Program: pjj0211.PAS / C / CPP
Batas Run-time: 1 detik / test-case
Batas Memori: 16 MB
Nama Berkas Masukan: Standard input (keyboard)
Nama Berkas Keluaran: Standard output (layar)

Buatlah program untuk menemukan faktor-faktor dari suatu bilangan bulat. Faktor-faktor suatu bilangan bulat N adalah bilangan-bilangan bulat positif yang habis membagi N. Misalnya, faktor dari 24 adalah 24, 12, 8, 6, 4, 3, 2, dan 1.

FORMAT MASUKAN

Baris pertama berisi sebuah bilangan bulat N (1 ≤ N ≤ 1000000).

FORMAT KELUARAN

Keluarkan faktor-faktor dari N, satu bilangan pada setiap baris, terurut dari besar ke kecil.
