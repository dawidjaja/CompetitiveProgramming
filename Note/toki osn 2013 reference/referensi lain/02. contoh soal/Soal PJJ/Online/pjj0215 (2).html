Prima Ke-K?

Nama Program: pjj0215.PAS / C / CPP
Batas Run-time: 1 detik / test-case
Batas Memori: 16 MB
Nama Berkas Masukan: Standard input (keyboard)
Nama Berkas Keluaran: Standard output (layar)

Dalam soal ini, yang perlu Anda lakukan adalah menjawab pertanyaan "Berapakah bilangan prima ke-K?". Anda akan diberikan masukan yang berisi N buah K, jawablah N pertanyaan tersebut.

FORMAT MASUKAN

Baris pertama berisi sebuah bilangan bulat N (1 ≤ N ≤ 20000). N baris berikutnya berisi masing-masing sebuah bilangan K (1 ≤ K ≤ 77777).

FORMAT KELUARAN

N baris berisi masing-masing sebuah bilangan yang merupakan jawaban dari "Berapakah bilangan prima ke-K?".

CONTOH MASUKAN 

4
1
3
5
2

CONTOH KELUARAN 

2
5
11
3

Penjelasan

Anda ditanya "Berapakah bilangan prima ke-1, ke-3, ke-5, dan ke-2?", tentu jawabannya adalah 2, 5, 11, dan 3. Untuk menyelesaikan soal ini, Anda mungkin perlu membaca dan mempelajari Teknik Eratosthenes Sieve.
